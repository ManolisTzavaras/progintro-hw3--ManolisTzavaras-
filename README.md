
Εισαγωγή
Καταρχάς, θα ήθελα να ξεκινήσω λέγοντας ότι το παιχνίδι Go είναι ένα εξαιρετικά δύσκολο παιχνίδι με υπερβολικά πολλές κινήσεις και συνδυασμούς. Είναι γνωστό στην επιστήμη των υπολογιστών ότι δεν είναι εύκολα εφικτό για ένα άτομο να γράψει κώδικα που να το επιλύει πλήρως ή να παίζει τέλεια, γι' αυτό και η προσέγγισή μας επικεντρώνεται στη δημιουργία μιας λειτουργικής μηχανής βασισμένης σε κανόνες.

Το βασικότερο που παρατήρησα σε αυτή την εργασία είναι ότι το Go έχει περισσότερους συνδυασμούς από το σκάκι. Αυτό ήταν κάτι που με φόβισε στην αρχή, καθώς έχω ξαναπροσπαθήσει να φτιάξω σκάκι στο παρελθόν και η διαχείριση της πολυπλοκότητάς του ήταν τεράστια πρόκληση.
